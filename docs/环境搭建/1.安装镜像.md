# 安装虚拟机
直接从官网下载vmware按照默认配置安装，注意内存尽量选大些
# 下载并选择镜像文件

# 新建虚拟机
1. `文件` -> `新建虚拟机` -> 选择iso镜像
1. 创建一个root用户 密码建议 123

# 虚拟机联网
- 博客：https://blog.csdn.net/dd2016124/article/details/128361941
- 如果在网络启动的时候报错可以看`tail -n 200 /var/log/messages|grep network`看日志排查
- 如果出现找不到动态库的情况通过`ldd "which dhclient"`看缺少哪个

# 虚拟机文件共享
- 博客：https://blog.csdn.net/rjszcb/article/details/128147795
- `vmhgfs-fuse .host:/ /mnt/hgfs`
